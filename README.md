# Finance Tracker

## Overview
Local, single-user finance tracker.
C++ backend with a web frontend.

## Architecture
- `backend/` – C++ finance engine + HTTP API
- `frontend/` – Web UI (tech TBD)

## Development
Backend builds on Linux (Debian).
Frontend runs locally and talks to localhost API.
